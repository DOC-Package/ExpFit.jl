ExpFit.espira1 — Method
espira1

This function first computes a modified FFT of y and places knots on the unit circle. It then calls the AAA routine from RationalFunctionApproximation.jl to obtain a rational approximant of (modified) DFT data, and finally extracts the ESPIRA parameters via a partial-fraction procedure.

source
ExpFit.espira2 — Method
espira2

This function computes a shifted FFT of its input data and uses AAA to obtain a rational approximant. It then extracts the exponential parameters by converting the resulting poles and residues into a sum of exponentials.
